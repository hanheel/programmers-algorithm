# DP
DP는 값을 계속해서 추가하고 연산하는 구조이다
백트래킹은 잘못된 경로를 추적하고 돌아오는 구조이다
= 어떤 선택을 했을 때 해당 선택이 실패하면 원래 상태로 돌아와서 다른 선택을 시도한다

연속된 숫자 인덱스이기 때문에 배열을 사용한다

7일 상담 : 불가능 (d[7] = 0)
6일 상담 : 불가능 (d[6] = 0)
5일 상담
  - 상담할 경우 최대 : 15 (d[5] = 15, 그 뒤는 상담 불가)
  - 상담 안할 경우 최대 : 0 (d[6])
4일 상담
  - 상담할 경우 최대 : 35 (d[4] = 35)
  - 상담 안할 경우 최대: 15 (d[5])
3일 상담
  - 상담할 경우 최대: 45 (d[3] = d[4] + 10 = 45)
  - 상담 안할 경우 최대: (d[4] = 35)
2일 상담
  - 상담할 경우 최대: 20 (자기자신만 가능)
  - 상담 안할 경우 최대: 45 (d[3])
1일 상담
  - 상담할 경우 최대: 55 (d[1] = d[4] + 10)
  - 상담 안할 경우 최대: 45 (d[3])


만약 상담 일수를 구하고 싶다면?
- DP로 최댓값을 구한 뒤, 그 값을 다시 역추적한다
예시
```bash

dp[8] = 0
dp[7] = 0
dp[6] = 0
dp[5] = 15
dp[4] = 35
dp[3] = 45
dp[2] = 45
dp[1] = 45
```
- 1일이 skip 됐다 = 2일이랑 값이 같다
- 2일이 skip 됐다 = 3일이랑 값이 같다
