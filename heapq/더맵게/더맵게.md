## 힙 - 우선순위큐

정렬 후 popleft가 필요한 것 같은데.. 큐는 정렬이 불가능하다 = 우선순위큐를 고려해본다

## 효율성 테스트

```python
heapq.nsmallest(n, iterable) # 가장 작은 n개 원소 반환
```

- 시간 복잡도 : O(n)
  heap에서는 [0] 요소가 제일 작다 = 인덱스를 사용할 것

```python
heap[0]
```

## 주의 사항

- 연산을 시작하기 전 모든 요소의 값이 K 이상인지 확인해야 한다
  모든 값이 K 이상이면 연산을 돌 필요가 없다 = 연산 횟수는 0이다
